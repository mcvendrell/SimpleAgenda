1.- Requerimientos mínimos para la aplicación:

	- Sistema Operativo Windows 9x, Windows NT.
	- 8 Mb de memoria RAM.
	- 8 Mb de espacio libre en el disco duro (más al menos 4 durante la instalación).
	- Ratón instalado.
	- Para la configuración regional de hora no usar el formato AM/PM

    NOTA: Si el programa da un error del tipo "no se ha encontrado xxx.OCX o falta algún componente..."
	  acuda a mi web http://manuel.conde.name y bájese la instalación del programa, la cual le añadirá
	  al sistema los componentes que necesita la aplicación para funcionar.


2.- Descripción:

Esta sencilla aplicación permite establecer avisos en cualquier fecha y a cualquier
hora del día con un consumo mínimo de CPU (menos del 0.01% bajo Windows NT 4.0).

Para ello emplea una base de datos creada en Access 97 de un tamaño muy pequeño
(80 Kb sin datos).

La aplicación no mantiene una conexión a la base de datos constantemente (esto consumiría
recursos) sino que se conecta para obtener los datos que necesita y se desconecta cuando
ya no precisa más datos. La aplicación se ha desarrollado con el menor consumo posible
de recursos. Una vez que se activa la agenda, por regla general nunca se encuentra conectado
a la base de datos (no consume recursos).

Dispone de una pequeña ayuda en el menú principal de la aplicación.
